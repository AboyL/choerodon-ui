## Usage

```jsx
import { configure } from 'choerodon-ui';

configure({ prefixCls: 'ant' });
```

## API

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| prefixCls | set prefix class | string | c7n |
